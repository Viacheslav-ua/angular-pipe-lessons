json - удобная отладка в шаблонах

{{ {name: 'Linsey', surname: 'Banach'} }}

{{ {name: 'Linsey', surname: 'Banach'} | json }}

{{ {name: 'Linsey', surname: 'Banach'} | json }}
Currency - валюта на сайте

{{ '555' | currency }}

{{ '555' | currency: 'UAH' }}

{{ '555' | currency: 'UAH': 'symbol-narrow' }}

{{ '555' | currency: 'EUR': 'symbol-narrow': '1.0-2' }}

Percent - вывод процентов из долей

{{ 1/3 }}

{{ 1/3 | percent }}

{{ 1/3 | percent: '1.0-3' }}

{{ 2.5 | percent: '2.5-6' }}

Number - форматирование вывода числа

{{ 10/3 }}

{{ 10/3 | number}}

{{ 10/3 | number: '1.0-2'}}

{{ 2.5 | number: '2.5-6'}}

Slice - получение среза строки или массива

{{ '123456789' | slice: 2 }}

{{ '123456789' | slice: 2: 5 }}

{{ '123456789' | slice: 2: -2 }}

{{ [1,2,3,4,5,6,7,8,9] | slice: 2 }}

{{ [1,2,3,4,5,6,7,8,9] | slice: 2: 5 }}

{{ [1,2,3,4,5,6,7,8,9] | slice: 2: -2 }}
